Site Plan: Sanj Corp 3D

Site Name

Sanj Corp 3D - Este nombre representa un emprendimiento boliviano dedicado a la impresión 3D personalizada. Es fácil de recordar y refleja el enfoque local y técnico.

Site Purpose

El sitio servirá como un portafolio para mostrar los trabajos realizados, ofrecer opciones de personalización, permitir a los usuarios solicitar cotizaciones y realizar pedidos de impresión 3D directamente desde la web.

Scenarios

  • ¿Qué tipo de filamento o materiales utilizan en las impresiones?
  • ¿Puedo subir mi archivo STL y obtener una cotización personalizada?

Color Schema

  • Primario: #0d6efd (Azul profesional, para encabezados y botones)
  • Secundario: #f8f9fa (Gris claro, para fondos y secciones)

Typography

  • Fuente principal: 'Roboto', sans-serif (para el contenido general)
  • Fuente secundaria: 'Montserrat', sans-serif (para títulos y encabezados)
